# telegram_bot



## perquisites:


-python 3.7

-a telegram account

-an OpenAI account for API keys 


## how to run a code:

```bash 
conda create -n telebot python = 3.7 -y
```



then activate this virtual env 

```bash
conda activate telebot
```
download requirements file 
```bash
pip install -r requirements.txt
```
